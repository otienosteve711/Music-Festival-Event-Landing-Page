# Music Festival Landing Page

A fully responsive event landing page built to showcase mastery of **CSS Grid**, **Flexbox**, **Bootstrap 5 Grid**, and **mobile-first responsive design**.  
This project demonstrates clean GitHub workflow practices and modern front-end development techniques.

---

## Features
- **Header Section with CTA**
  - Full-width hero section
  - Event name, tagline, and Bootstrap-styled CTA button
  - Flexbox alignment for vertical/horizontal centering
  - Gradient background (or image option)

- **Agenda Section**
  - Built with **CSS Grid**
  - Mobile-first single column, expands to multi-column on larger screens

- **Speakers Section**
  - Structured with **Bootstrap 5 Grid**
  - Responsive speaker cards with images and bios

- **Registration Section**
  - Flexbox alignment
  - Prominent CTA button styled with Bootstrap utilities

- **Footer**
  - Simple, full-width branding

---

## Tech Stack
- **HTML5**
- **CSS3 (Grid + Flexbox)**
- **Bootstrap 5**
- **Mobile-first responsive design**
